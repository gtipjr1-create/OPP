How to generate PNGs
# 1. Copy opp-icon.svg to your project root
# 2. Install sharp (one time)
npm install sharp

# 3. Run the generator
node generate-icons.mjs

# 4. Output goes to public/icons/
#    icon-192.png
#    icon-512.png
#    apple-touch-icon.png
#    favicon-32.png
#    favicon-16.png

# 5. Copy manifest.webmanifest to public/
# 6. Update layout.tsx icons to point to /icons/
